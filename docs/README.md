```
docs/
├── frontend/          # 前端页面逻辑文档
│   ├── README.md      # 前端架构说明
│   ├── 股票列表页.md
│   ├── K线图表页.md

│

    ├── README.md      # 后端架构说明
```

## 前端页面

基于 `index.html`，前端包含以下功能模块：

1. **股票列表** - 显示所有可用股票
2. **搜索功能** - 搜索股票代码或名称
3. **K线图表** - 使用 ECharts 展示 K 线数据
4. **数据导出** - 支持 CSV/JSON 导出
