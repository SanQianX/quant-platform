# 股票接口文档

## 模块: stock.py

### 功能说明
提供股票列表查询、搜索、K线数据获取等核心功能。

### 主要接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 股票列表 | GET | `/api/stock/list` | 获取所有股票 |
| 搜索股票 | GET | `/api/stock/search` | 搜索股票 |
| K线数据 | GET | `/api/stock/{code}` | 获取K线数据 |
| 历史K线 | GET | `/api/stock/{code}/history` | 获取历史K线 |

### 依赖服务
- `StockService` - 股票服务层
- `cache` - 缓存工具
- `logger` - 日志工具

### 数据流程
```
请求 → API层 → Service层 → 数据库/数据源 → 缓存 → 响应
```

### 相关配置
- `STOCK_LIST` - 股票列表配置
- `BASE_PRICES` - 基础价格
- `MOCK_DATA_CONFIG` - 模拟数据配置
