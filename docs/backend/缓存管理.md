# 缓存管理接口文档

## 模块: cache.py

### 功能说明
提供缓存管理功能，用于加速数据查询。

### 缓存策略

| 缓存键 | 数据类型 | TTL (秒) |
|--------|---------|---------|
| `stock:list` | 股票列表 | 3600 |
| `kline:{code}` | K线数据 | 300 |
| `kline:{code}:weekly` | 周K线 | 1800 |
| `kline:{code}:monthly` | 月K线 | 3600 |

### 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| 缓存状态 | GET | `/api/cache/stats` |
| 清空缓存 | POST | `/api/cache/clear` |

### 实现方式
- 内存缓存 (当前)
- Redis 缓存 (可扩展)
