# 定时任务接口文档

## 模块: scheduler.py

### 功能说明
管理定时数据采集任务，支持手动触发和自动调度。

### 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| 任务状态 | GET | `/api/scheduler/status` |
| 启动任务 | POST | `/api/scheduler/start` |
| 触发更新 | POST | `/api/scheduler/trigger` |
| 更新单股 | POST | `/api/scheduler/update/{code}` |

### 任务类型

| 周期 | 说明 |
|------|------|
| daily | 日线数据 |
| weekly | 周线数据 |
| monthly | 月线数据 |

### 数据源
- AkShare - 免费金融数据
- Tushare - 付费金融数据 (需要 Token)
