# 技术指标接口文档

## 模块: indicators.py

### 功能说明
提供各类技术指标计算，如 MACD、KDJ、RSI、布林带等。

### 支持的指标

| 指标 | 说明 |
|------|------|
| MACD | 指数平滑异同移动平均线 |
| KDJ | 随机指标 |
| RSI | 相对强弱指标 |
| BOLL | 布林带 |
| MA | 移动平均线 |
| EMA | 指数移动平均线 |

### 接口示例

```bash
# 获取MACD指标
GET /api/indicators/macd?code=600519

# 获取KDJ指标
GET /api/indicators/kdj?code=600519
```

### 计算逻辑
技术指标基于K线数据计算，使用 Pandas 进行数据处理。
