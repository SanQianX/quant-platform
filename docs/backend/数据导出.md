# 数据导出接口文档

## 模块: export.py, export_enhanced.py

### 功能说明
提供数据导出功能，支持多种格式导出股票数据。

### 支持格式

| 格式 | 说明 |
|------|------|
| CSV | 逗号分隔值文件 |
| JSON | JSON 格式 |
| Excel | Excel 文件 (增强版) |

### 接口

```bash
# 导出CSV
GET /api/export/600519?format=csv

# 导出JSON
GET /api/export/600519?format=json
```

### 导出字段
- 日期 (date)
- 开盘价 (open)
- 最高价 (high)
- 最低价 (low)
- 收盘价 (close)
- 成交量 (volume)
